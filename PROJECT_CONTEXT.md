## 1. 프로젝트 개요

**서비스명:** Current (by Jyos)
- 브랜드: `Current` (메인) / `Jyos` (사업자, 서브 브랜드)
- 사업자명: Jyos (개인사업자 기보유)
- 로고: 상어 지느러미 심볼 + "Current" 텍스트 조합

**한 줄 소개:**
외국 AI 모델 뉴스를 자동 수집·정리해 카드형 UI로 보여주고, 사용자가 원하는 모델의
**뉴스·주가를 ESP32 하드웨어 디스플레이로 받아보는** 서비스.
나아가 외국 주요 기업들의 **뉴스·주식까지 분석·큐레이션**해 하드웨어와 결합한다.

**배경 및 문제 정의:**
AI 기술의 발전 속도가 너무 빠르다. Claude, ChatGPT, Gemini 등 주요 AI 모델의 업데이트가
수개월 간격으로 쏟아지고, 관련 기사도 매일 수백 개씩 나온다. 그러나 이 정보를 한눈에
모아볼 수 있는 도구가 없어서 일일이 직접 찾아다녀야 한다. Current는 이 문제를 푸는
"AI 모델 정보 허브"에서 출발해, 사용자가 관심 있는 대상(AI 모델 → 더 나아가 글로벌 기업)의
**뉴스와 주가를 소프트웨어 + 전용 하드웨어로 동시에 소비**하게 만드는 것을 목표로 한다.

**최종 목표:**
- 졸업 프로젝트로 제출 (컴퓨터공학부, 개발 기간 9개월)
- 실제 운영 서비스로 론칭하여 수익화까지 달성
- 광고(Google AdSense) + 구독(₩2,000/월) + 하드웨어 판매 3가지 수익 모델

## 2. 핵심 기능 — 기본 목표 / 추가 목표

이 프로젝트는 두 트랙으로 나뉜다.
**기본 목표(반드시 완성)** 는 "AI 모델 뉴스 + 주가 → 하드웨어",
**추가 목표(여력 시 확장)** 는 "글로벌 기업 뉴스·주식 분석 → 하드웨어"이다.

### 🎯 기본 목표 (Core) — AI 모델 × 뉴스 × 주가 × 하드웨어

#### B1. AI 모델 뉴스 대시보드 *(현재 ~완성)*
- Claude / ChatGPT / Gemini 등 주요 AI 모델 관련 뉴스 RSS 자동 수집
- Claude API(Haiku) 기반 분류 + 한국어 요약 파이프라인
- 카드형 UI, 카테고리 필터, 모델별 페이지/상세 모달
- 별표(즐겨찾기) 저장 — 사용자별 북마크(Google 로그인 + Supabase)
- Cron Job(GitHub Actions)으로 자동 수집·분류

#### B2. AI 모델 ↔ 관련 기업 주가 연동
- 각 AI 모델을 모회사/상장사에 매핑 (예: ChatGPT→MSFT, Gemini→GOOGL, Copilot→MSFT 등)
- Yahoo Finance(yfinance) 등으로 해당 기업 주가·차트 데이터 수집
- 모델 페이지/상세에 "관련 기업 주가" 위젯 표시
- 비상장 모델(Anthropic 등)은 투자사/파트너사 또는 "비상장" 안내로 처리

#### B3. ESP32 하드웨어 디스플레이 *(기본 목표의 최종 산출물)*
- ESP32 + 4인치 TFT 디스플레이 (ILI9341 또는 동급)
- Wi-Fi로 백엔드 API에서 데이터 수신
- **사용자가 별표한(선택한) AI 모델의 헤드라인 + 관련 기업 주가를 디스플레이에 표시**
- 디바이스 ↔ 계정 페어링(선택 모델 동기화), 주기적 폴링으로 갱신
- Fusion 360 케이스 설계 + 3D 프린팅
- 한글 폰트 처리 주의 (비트맵 폰트 또는 우선 영문 헤드라인으로 구현)

### ➕ 추가 목표 (Extended) — 글로벌 기업 × 뉴스 분석 × 주식

#### E1. 외국 주요 기업 뉴스 수집·분석
- 빅테크 외 글로벌 주요 상장사(Apple, NVIDIA, Tesla, Amazon 등)로 뉴스 범위 확장
- 기업별 뉴스 수집 + Claude API로 **요약·감성(호재/악재) 분석·핵심 키워드 추출**
- 기업 단위 카드/페이지 (이미 보유한 171개 회사 로고 에셋 활용)

#### E2. 기업 주식 분석
- 기업 뉴스와 주가를 연계해 "뉴스 → 주가 영향" 관점으로 묶어 표시
- 주가 차트 + 뉴스 타임라인 오버레이, 간단한 추세/지표
- (졸업 발표 데모 수준 → 운영 단계에서 고도화)

#### E3. 기업 뉴스·주가 하드웨어 결합
- B3 하드웨어를 확장해, AI 모델뿐 아니라 **선택한 글로벌 기업의 헤드라인·주가**도 표시
- 사용자가 "내 관심 대상"(AI 모델 + 기업)을 골라 디바이스에 띄우는 통합 경험

> **참고 — Phase 2 글로벌 뉴스레터(미국/유럽 경제, 한국 정치 등)**
> 기존 계획의 범용 글로벌 뉴스레터 확장은 **보류/축소**한다. 방향을 "기업·주식 중심 분석"으로
> 좁혀 정체성을 분명히 하기 위함. (NewsAPI / 네이버 뉴스 RSS는 필요 시 E1에서 재활용)

## 3. 수익화 모델

**광고:**
- Google AdSense: 양쪽 사이드바 광고 (무료·유료 유저 모두에게 표시)
- 금융·투자(기업 주식) 카테고리 기사가 있어 광고 단가 유리

**구독 (₩2,000/월):**
- 광고 제거가 아닌 추가 기능 제공 방식
- 주가 연동, 기업 분석, 맞춤 피드, 하드웨어 다중 디바이스 연동 등
- 결제: Stripe (글로벌) / 토스페이먼츠 (한국)

**하드웨어 판매:**
- ESP32 + TFT 디스플레이 완제품(케이스 포함) 판매
- 구독과 연동해 "디바이스 + 프리미엄 데이터" 번들 가능

**인프라:**
- Vercel 무료 배포로 시작
- 유저 증가 시 AWS(EC2, S3, RDS, CloudFront)로 이전

## 4. 기술 스택

### 프론트엔드
- **Next.js 16** (App Router) — SSR, SSG, ISR 모두 활용, /app 디렉토리 기반 라우팅
- **TypeScript** — strict 모드, 모든 파일 .tsx / .ts
- **Tailwind CSS v4** — utility-first 스타일링, 별도 CSS 파일 최소화
- **framer-motion** — 인터랙션/애니메이션

### 백엔드 (Next.js 내부)
- **Next.js Route Handler** (app/api/) — RESTful API 설계
- **Node 크롤러** (`scripts/crawl-articles.mjs`) — rss-parser 기반 수집·분류
- Cron Job: GitHub Actions(6시간 주기) — 추후 Vercel Cron 검토

### 데이터베이스 / 인증
- **Supabase** — PostgreSQL, Auth + Google OAuth, RLS, (필요 시) Realtime
- **Redis** (선택, 추후) — API 응답 캐싱, 주가/요약 중복 방지

### AI / 외부 API
- **Claude API** (Anthropic) — 기사 분류(Haiku) + 요약 + (E1) 기업 뉴스 감성·키워드 분석
- **주가 데이터** — Yahoo Finance(yfinance) 등 (AI 모델 관련 기업 → 글로벌 기업으로 확장)
- **NewsAPI.org / 네이버 뉴스 RSS** — (필요 시) 기업 뉴스 소스 보강

### 하드웨어
- **ESP32** — Wi-Fi, C++ (Arduino), ArduinoJson, HTTPClient
- **4인치 TFT 디스플레이** — ILI9341, TFT_eSPI 라이브러리
- **3D 프린팅 케이스** — Fusion 360 설계
- 백엔드 연동: 디바이스 전용 읽기 API(선택한 모델/기업의 헤드라인·주가) + 페어링

### 배포 / 인프라
- **Vercel** — 프론트엔드 + API 배포, 환경변수, Cron Jobs
- **GitHub** — 버전 관리, GitHub Actions CI/CD (크롤러 cron 가동 중)
- **AWS** (스케일 시) — EC2, S3, RDS, CloudFront
